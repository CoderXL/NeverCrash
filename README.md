# NeverCrash
一个无侵入的 iOS crash 防护框架

## 功能
- unrecognized selector crash
- KVO crash
- NSNotification crash
- NSTimer crash
- Container crash
- NSString crash
- Bad Access crash
- UI not on Main Thread crash
- navigation crash

## 原理
可以参考[网易Baymax](https://neyoufan.github.io/2017/01/13/ios/BayMax_HTSafetyGuard/)
